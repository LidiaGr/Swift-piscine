# Swift-piscine DAY-03

## Photo / Gallery App
Simple Application that downloads pictures from the Internet using Threads.
Application send Alert, when download failed.
Pictures can be shown fully-sized, zommed and scrolled.

Learned:
- CollectionView
- Multithreads
- Alerts
- ScrollView, Zoom
